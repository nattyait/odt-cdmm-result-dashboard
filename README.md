# odt-cdmm-result-dashboard Skill

Create fact-based CDMM assessment dashboards in minutes.

## Quick Start

**Organization:** [Your Company]
**Mode:** A / B / C
**Team:** [Team Name]
**Assessment URL:** https://cdmm.odt.co.th/group-assessment/[ID]/report

See SKILL.md for detailed instructions and examples.

## Features

- **Mode A:** Single team analysis
- **Mode B:** Multiple teams comparison dashboard
- **Mode C:** Add team to existing dashboard
- **Fact-based analysis** — no assumptions, observable data only
- **Interactive HTML** — responsive, mobile-friendly
- **ODT training links** — recommended courses with direct links
- **CDMM definitions** — references InfoQ article definitions

## How to Use

1. Tell the skill your organization name
2. Choose your mode (A, B, or C)
3. Provide team name(s) and assessment URL(s)
4. Receive your live dashboard HTML file

## Requirements

- Internet access to fetch CDMM assessment reports from https://cdmm.odt.co.th/
- Assessment report URL from your CDMM platform

## Training

All training course recommendations come from: https://training.odt.co.th/

See SKILL.md for complete documentation, examples, and quality standards.
